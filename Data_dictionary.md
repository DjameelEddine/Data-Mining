## Data dictionary for packages dataset

**RECPTCL_FID** : Receptacle ID (Container) The ID of the bag or sack that contains this specific parcel

**MAILITM_FID** : Mail Item ID (Parcel) The unique tracking number of the parcel

**date** : The exact date and time the event happened

**etablissement_postal**: Current Location The code of the facility where the event occurred

**EVENT_TYPE_CD**: event code indicating what happened

**next_etablissement_postal**: Next Hop / Destination

## Data dictionary for receptacle dataset
**RECPTCL_FID** : Receptacle ID The unique ID of the bag itself.

**date** : The exact date and time the event happened

**etablissement_postal** : Current Location Where the bag is right now.

**EVENT_TYPECD** : Event Code same as above

**nextetablissement_postal** : Next Hop / Destination
